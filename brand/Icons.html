brand-kit / icon-set v1.0 · 26 May 2026

A uniform icon system.

~250 icons across 16 categories — cloud, AWS, Azure, M365, ITSM, DevOps, identity, AI, maps and more. Three render modes (line / gradient / badge). Click any icon to copy its SVG to the clipboard, ready to drop straight into a slide, page, or doc. All on a 24px grid with consistent stroke — they'll sit cleanly alongside `~bmc` type, the brand gradient and the rest of the system.

— icons

// Drop into anything · Word · PowerPoint · Pages · Keynote · Web

PowerPoint SVG
→ Insert · Pictures · This Device · pick the .svg
→ right-click · Convert to Shape to recolour
Keynote SVG
→ drag the .svg onto a slide
→ Format · Image · Replace swaps it
Word SVG
→ Insert · Pictures · From File
→ Microsoft 365: tints follow the text colour
Pages SVG
→ drag the .svg into the canvas
→ resize at any scale (vector)
Web / Email PNG
→ paste the SVG inline in HTML
→ or use the 512 PNG for email signatures

// usage

// copy an icon, paste into your slide / page, then style:
<svg ... class="bmc-icon">...</svg>

.bmc-icon {
  width: 24px; height: 24px;
  stroke: currentColor;       /* tints from text colour */
  stroke-width: 1.5;
  fill: none;
}

/* tint with the brand gradient */
.bmc-icon--grad { stroke: url(#bmc-grad); }
/* drop into a badge */
.bmc-icon--badge { color: #fff; background: var(--bmc-gradient); padding: 8px; border-radius: 8px; }

// Tip: include the <linearGradient id="bmc-grad"> definition once at the top of any page that uses the gradient mode. The defs block at the top of this file is reusable verbatim.
