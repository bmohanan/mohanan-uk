// 04 — Rules of the C

marks/c/rules
  • + DO
    Always render the C as gradient letterform — never as a solid colour.
  • + DO
    Pair with ~bmc on the opposite corner so the system reads at a glance.
  • − DON'T
    Don't outline, fill-in, or trace a box around the C. The point is the box is gone.
  • − DON'T
    Don't use the C standalone where the audience doesn't already know "BMC" — use the lockup instead.
